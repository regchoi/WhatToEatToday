# 오늘 뭐 먹지? (음식 추천 서비스 사이트)

## 프로젝트 개요
오늘의 음식을 추천해드립니다.</br>
여러분의 기분, 날씨, 상황에 따라 여러 음식을 준비해보았습니다.</br>


## 선행 작업
1. H2 설치하기
2. Github Repository clone

## 실행 방법
1. application.yml 파일에 ddl-auto: create로 변경
2. 프로젝트 안에 initdb.java 주석 풀고 실행
3. 실행 후 application.yml 파일에 ddl-auto: none으로 변경
4. initdb.java 주석처리
